# `מָשְׁכֵ֖נִי ונָּר֑וּצָה` | `Pull & Run`

    מָשְׁכֵ֖נִי אַחֲרֶ֣יךָ נָּר֑וּצָה
    הֱבִיאַ֨נִי הַמֶּ֜לֶךְ חֲדָרָ֗יו
    נָגִ֤ילָה וְנִשְׂמְחָה֙ בָּ֔ךְ
    נַזְכִּ֤ירָה דֹדֶ֙יךָ֙ מִיַּ֔יִן
    מֵישָׁרִ֖ים אֲהֵבֽוּךָ
